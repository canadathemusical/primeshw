Each Line

This is pretty standard code stuff here. Let's go over it line by line:

  1. m=prompt("number?")
    Here we declare the variable "m" and make it equal to whatever a user enters in a prompt. The prompt will appear on the screen reading "number?" just like the one you will encounter when you click the button at the end of this document. So, if the user enters "1", m = 1. 

  2. for (n=2;n<=m;n=n+1)
    This is what is called a "for loop". It's a pretty standard way to loop or iterate in programming. It is usually the first way someone learns to do such a thing. It breaks down into a few parts: 
    • n=2;
      This is the starting point. Our loop starts at 2
    • n<=m;
      While n is less than or equal to m, keep going.
    • n=n+1
      At each iteration, go up by 1. So, our loop is like 2, 3, 4, 5, 6, 7, 8.. m. This can also be written as: 
      n++
      or 
      n+=1

  3. {
    This opening bracket or brace starts the block for the loop we discussed above. Your professor's code is unindented, and genrally poorly formatted so it's difficult to read and see where braces do anything. At the end, I will write a more standard way of writing this code. 

  4. var flag=0
    Here we decalre the variable "flag" and set it to 0. 

  5. for (i=2;i<=Math.sqrt(n);i=i+1)
    Opening another for loop... a for loop inside of a for loop. We are starting to get complex. 
    • i=2;
      "i" is equal to 2 so we start at 2
    • i<=Math.sqrt(n);
      While "i" is less than or equal to the square root of n, do a thing
    • i=i+1
      Go up by one at each iteration.

  6. {
    This brace opens the second for loop. 

  7. if(n%i==0){flag=1}
    Here with a have a condition. A condition in a for loop in a for loop. The mind simply boggles over this complexity. IF something evaluates to "true", do something. In this case, a way to say it in plain english is "if n is divisible by i, flag equals 1". So that condition is a great way to find whether something is divisible by something else. It's called modulo, that %. It gets the remainder of a division. 3%2=1. To find even numbers we do 
    if(someInt%2==0){console.log(`${someInt} is fucking even!`)}
    . Another odd programming thing is in this line. The "==" is is a comparison operator. If you write "=", you're declaring a variable or cosntant. "==" and "===" compare two items. You can use "==" for now as "===" gets more tricky. 

  8. }
    This closing brace ends the second for loop.. finally. 

  9. if(flag==0){console.log("prime",n)}
    Another condition followed by a console.log. If "flag" equals 0, print "prime" followed by the value of n to the console. 

  10. }
    This closing brace or bracket ends the first for loop and the entire program. After this, things will print to the console as desired.

Break It Down

Unfortunately, the line by line approach doesn't really give a great picture of anything more than JS syntax. Let's peel this thing apart into separate functions. That way we can really figure out how it works. starting from the center out.

  1. var flag=0
for (i=2;i<=Math.sqrt(n);i=i+1) {
	if(n%i==0) {
		flag=1
	}
}

    This does the main work of finding primes. We can move that prime finder into a separate function for clairty and reusability.

    
const isPrime = (n) => {
	if (n < 2) return false;
	for (i=2;i<=Math.sqrt(n);i=i+1) {
		if(n%i==0) {
			return false;
		}
	}
	return true;
}
    Now, we have a function that returns true if n is prime. 
  2. 
for (n=2;n<=m;n=n+1)
{
	//our other for loop was here.
if(flag==0){console.log("prime",n)}
}
    This piece of the original puzzle counted from 2 (the first prime) to the input number and spat out data for each prime flagged (or unflagged actually). 
    
const printConstituentPrimes = (m) => {
	for (n=2;n<=m;n=n+1) {
		if(isPrime(n)){console.log("prime",n)}
	}
}
    Now we're cookin with chicken grease. We invoked the isPrime function to log any primes we count above 2 and up to m. We can call that function now without the prompt or with it or however we want because we just pass in the arguments between the ().

	let m = prompt("number?");

const isPrime = (n) => {
	if (n < 2) return false;

	}
}
