this is pretty standard code stuff here. let's go over it line by line:

  1. m=prompt("number?")
    here we declare the variable "m" and make it equal to whatever a user enters in a prompt with will appear on the screen reading "number?" just like the one you will encountered when you the button at the end of this document. so, if the user enters "1", m = 1. 
  2. for (n=2;n<=m;n=n+1)
    this is what is called a "for loop". it's a pretty standard way to loop or iterate in programming. it is however archaic, but usually the first way someone learns to do such a thing. it breaks down into a few parts: 
    • n=2;
      this is the starting point. our loop starts at 2
    • n<=m;
      while n is less than or equal to m, keep going
    • n=n+1
      at each iteration go up by 1. so out loop is like 2, 3, 4, 5, 6, 7, 8,.. m. This can also be written as 
      n++
      or 
      n+=1
  3. {
    this opening bracket or brace starts the block for the loop we discussed above. Your professor's code is unindented, and genrally poorly formatted so it's difficult to read and see where braces do anything. at the end, I will write a more standard way of writing this code. 
  4. var flag=0
    here we decalre the variable flag and set it to 0. 
  5. for (i=2;i<=Math.sqrt(n);i=i+1)
    opening another for loop. a for loop inside of a for loop. 
    • i=2;
      i is equal to 2 so we start at 2
    • i<=Math.sqrt(n);
      while i is less than or equal to the square root of n, do a thing
    • i=i+1
      go up by one at each iteration.
  6. {
    this brace opens the second for loop. 
  7. if(n%i==0){flag=1}
    Here with a have a condition. IF something evaluates to "true" do something. in this case a way to say it in plain english is "if n is divisible by i, flag equals 1". so that condition is a great way to find whether something is divisible by something else. It's called modulo, that %. it gets the remainder of a division. 3%2=1. to find even numbers we do 
    if(someInt%2==0){console.log(`${someInt} is fucking even!`)}

  8. }
    this closing brace ends the second for loop 
  9. if(flag==0){console.log("prime",n)}
    another condition followed by a console.log. If flag equals 0, print "prime" followed by the value of n to the console. 
  10. }
    this closing brace or bracket ends the first for loop and the entire program. after this, things will print to the console as desired.
